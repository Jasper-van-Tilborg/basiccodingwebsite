Introductie tot CSS

CSS (Cascading Style Sheets) is de opmaaktaal voor het vormgeven van webpagina's. Het zorgt voor de visuele presentatie van de inhoud, zoals kleuren, lettertypen, marges en achtergronden. Met CSS kun je de lay-out en stijl van een website volledig aanpassen en zo een unieke uitstraling creÃ«ren.

Hoe werkt CSS

CSS werkt door middel van regels, die worden toegepast op HTML-elementen. Elke regel bestaat uit een selector en een set van eigenschappen en waarden. De selector bepaalt op welke elementen de regel van toepassing is, terwijl de eigenschappen en waarden aangeven hoe deze elementen moeten worden vormgegeven.
